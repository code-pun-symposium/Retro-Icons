# Retro-Icons

Sharing cute and retro 90s' & y2k icons downloaded from **[Iconfactory](http://iconfactory.com/)**.

Please use according to license!

## How to Use

1. Download `.dmg` file

2. Open `.dmg`, and you can see folders with icons in `/icons`

3. Use these folder :)
